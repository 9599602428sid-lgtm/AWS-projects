# Static Portfolio Website — AWS S3 + CloudFront

## Overview
A personal portfolio website hosted on AWS S3 with CloudFront CDN for global delivery.

## AWS Services Used
- S3 — Static file hosting
- CloudFront — Global CDN + HTTPS
- Route 53 — Custom domain DNS (optional)
- ACM — Free SSL certificate (optional)

## Architecture
Browser --> CloudFront --> S3 Bucket
